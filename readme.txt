== Installation ==

Install HM Testimonial as a regular WordPress plugin. Here are different ways to install plugin:

**Install by search plugin:**

1. In your Admin, go to menu Plugins > Add New
2. Search HM Testimonial
3. Click to install
4. Activate the plugin
5. Setup the proper settings in HM Testimonial menu

**Install by upload plugin:**

1. Download the latest version of HM Testimonial (.zip file)
2. In your Admin, go to menu Plugins > Add New
3. Select "Upload Plugin"
4. Click on "Choose File"
5. Select downloaded hm-testimonial.zip and click on "Install Now" button
6. Activate the plugin
